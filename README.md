# PCB do Painel Solar 1U — ICARUS

Repositório de trabalho relativo ao desenho preliminar da PCB do painel solar 1U do projeto ICARUS, no âmbito da Porto Space Team.

## Conteúdo
- Projeto KiCad da PCB;
- Footprint preliminar da célula solar TJ3xSCA-4080 / TJ30SCA-4080 CIC;
- Footprint provisório da interface de saída do painel;
- Notas de revisão técnica.

## Estado do desenvolvimento
A versão atual constitui uma base de revisão interna. O desenho representa:
- duas células por face;
- orientação das abas para o exterior;
- configuração elétrica em série 2S;
- zona de ligação ao EPS.

Antes de fabrico, devem ser confirmados:
1. conector e pinout finais;
2. polaridade e geometria dos contactos da célula;
3. compatibilidade mecânica com a estrutura;
4. margem de tensão face ao EPS;
5. validações ERC/DRC finais.
